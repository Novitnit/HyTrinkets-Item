# HyTrinkets-Item

HyTrinkets-Item is a plugin designed as an example for HyTrinkets-Core
This plugin adds custom items to the system.

## Added Item

#### Resurrection Collar
 Resurrection Collar is a necklace item that grants the player a second chance upon death.
 When the player dies:
 - The player will be revived with 1 health point.
 - The player will receive healing for 10 seconds.
